# Python To-Do App

A simple to-do list application built in Python with three different interfaces: Command Line (CLI), Desktop GUI (FreeSimpleGUI), and a Web App (Streamlit). All versions share the same backend and store tasks in a text file.

## Features

- Add tasks
- Edit tasks
- Complete (remove) tasks
- Automatically save tasks
- CLI, Desktop GUI, and Web versions

## Technologies

- Python
- FreeSimpleGUI
- Streamlit

## Installation

```bash
pip install FreeSimpleGUI streamlit
```

## Run the Project

### CLI

```bash
python cli.py
```

### Desktop GUI

```bash
python gui.py
```

### Web App

```bash
streamlit run web.py
```
